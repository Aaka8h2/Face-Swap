## üöÄ How to Run Locally

1.  **Clone this repository:**
    ```bash
    git clone https://github.com/Aaka8h2/Face-Swap.git
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the application:**
    ```bash
    python app.py
    ```

4.  **Open in Browser:**
    Go to `http://localhost:5000`
